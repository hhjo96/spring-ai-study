기능 안내

SafeGuard Advisor

민감 단어(욕설, 폭력 등) 포함 시 요청을 자동 차단합니다.

대화 기억 (JDBC)

최근 50개 메시지를 PostgreSQL에 저장하여 멀티턴 대화를 유지합니다. 서버 재시작 후에도 기억이 남습니다.

시맨틱 검색

질문과 유사한 도서를 VectorStore에서 자동 검색하여 LLM에 컨텍스트로 제공합니다.

로깅 Advisor

모든 요청/응답이 서버 DEBUG 로그에 기록됩니다.

시맨틱 도서 검색

자연어로 검색하면 의미적으로 유사한 도서를 VectorStore에서 찾아줍니다.

상세 검색 옵션

새 도서 등록

RDB에 저장하고 VectorStore에 임베딩하여 즉시 시맨틱 검색이 가능합니다.

도서 탐색

PostgreSQL에 저장된 도서 목록입니다.

임베딩 실험실

텍스트를 벡터로 변환하고 임베딩 모델의 출력을 확인합니다.
